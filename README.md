静态博客、个人简历和一些乱七八糟的作品

##### How to use...
* ###### Installation
    install dependencies:

    `npm i`

    run for dev:

    `node ./index.js` or `npm run go:dev`

    run for prod:

    `npm run go`

* ###### Vuepress build
    use `yarn` :

    `yarn blog:build`

* ###### build & publish
    Windows PowerShell:

    `build.bat`

    Linux bash:

    `sh build.sh`

then enjoy it~
